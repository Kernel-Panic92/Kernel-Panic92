### 🚧 Proyecto actual: Horix ERP

Plataforma ERP modular con autenticación centralizada, RBAC y orquestación vía MCP Gateway.

```
         Nginx SSL
        ┌────┴────┐
        │         │
   Shell SPA   MCP Gateway
   (login,     ├── Auth / JWT
    admin UI)  ├── RBAC centralizado
               ├── Proxy API a módulos
               ├── MCP tools (Claude)
               │
        ┌──────┼──────┐
        │      │      │
     Horix   DocFlow  (CRM→)
    HH.EE.   Facturas
    Nómina   Proveed.
```

**Stack:** Node.js, Express, SQLite, PostgreSQL, JWT, Nginx  
**Principios:** Sin código duplicado, permisos centralizados, módulos delgados
